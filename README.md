# Car Agency Web Application

## Description
A full-stack web application built with ASP.NET and SQL Server for managing a car agency system.

## Features
- Admin panel to add, update, and manage cars
- User panel to browse and simulate purchasing cars
- Filter cars by company
- Database integration using SQL Server
- Built using OOP principles

## Technologies
- C#
- ASP.NET
- SQL Server
